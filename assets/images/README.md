# Иконка приложения

Поместите файл иконки карандаша с именем `app_icon.png` в эту папку.

Требования к иконке:
- Формат: PNG
- Размер: 1024x1024 пикселей (рекомендуется)
- Фон: прозрачный или белый
- Форма: квадратная

Вы можете скачать иконку карандаша с одного из этих сайтов:
- https://www.flaticon.com/search?word=pencil
- https://icons8.com/icons/set/pencil
- https://www.iconfinder.com/search?q=pencil

После добавления файла `app_icon.png` выполните:
```bash
flutter pub get
flutter pub run flutter_launcher_icons
```
